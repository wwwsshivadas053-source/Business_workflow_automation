A modern AI-ready Business Process Automation platform built with Flask to digitize and streamline common organizational workflows. The application centralizes employee operations through secure authentication, role-based dashboards, automated workflows, analytics, and document management.

# ⚡ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/business-workflow-automator.git
```

Enter project

```bash
cd business-workflow-automator
```

Create virtual environment

```bash
python -m venv venv
```

Activate

Windows

```bash
venv\Scripts\activate
```

Linux

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```
